1. Create a folder
mkdir folderName
cd folderName

2. Create index.html
nano index.html

Paste:

<!DOCTYPE html>
<html>
<head>
    <title>My Docker HTML Page</title>
</head>
<body>
    <h1>Hello from Docker + Nginx!</h1>
</body>
</html>

3. Create Dockerfile
nano Dockerfile

Paste:

FROM nginx:latest
COPY index.html /usr/share/nginx/html/index.html

4. Build image
sudo docker build -t code .

5. Run container
sudo docker run -d -p 8090:80 code

6. Open in browser

http://localhost:8090/index.html
